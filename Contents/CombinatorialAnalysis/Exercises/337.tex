هریک از روابط زیر را اثبات کنید:
\begin{enumerate}
\item
$$m!n! | (m + n)!$$
\item
$$\binom m m + \binom{m+1}{m} + \binom{m+2}{m} + \cdots + \binom{m+n}{m} = \binom{m+n+1}{m+1}$$
\item
$${\binom{n}{0}}^2 + {\binom{n}{1}}^2 + {\binom{n}{2}}^2 + \cdots + {\binom{n}{n}}^2 = {\binom{2n}{n}}^2$$
\end{enumerate}
