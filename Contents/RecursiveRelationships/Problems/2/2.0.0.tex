    $$x G(x) = \sum_{k=0}^{\inf} a_k x^{k+1} = \sum_{k=1}^{\inf} a_{k-1}x^{k}$$
    $$x^2G(x) = \sum_{k=0}^{\inf} a_k x^{k+2} = \sum_{k=2}^{\inf} a_{k-2}x^{k}$$
    $$x^3G(x) = \sum_{k=0}^{\inf} a_k x^{k+3} = \sum_{k=3}^{\inf} a_{k-3}x^{k}$$

    $$= a_0 + a_1x + a_2x^2 - 5(a_0x + a_1x^2) + 8(a_0x^2) + $$
